Model View Pizza

A practical application of JavaScript MV* Frameworks

AngularJS 1

AngularJS is a highly productive MVC framework by Google that has widespread support by the developer

Kendo UI

Kendo UI is a web controls framework that can be used in the MVVM style. An open source version was released in 2014 with a limited set of controls

KnockoutJS

KnockoutJS is a MVVM framework that is simple and explicit. It was released in 2010 and has stayed very easy to use and easy to understand

RactiveJS

RactiveJS is a template-driven UI library that transforms inert HTML templates into blueprints for interactive apps.

ReactJS

ReactJS is a powerful view engine framework by Facebook that has widespread support by the developer community

VueJS

VueJS is a highly structured MVVM framework that prescribes how to build modular web applications using newer features of JavaScript
